In order to take the negative of a picture, you simply need to change the color of each pixel in the following way: Change each one of the three color component values to its complement with respect to the maximum value of 255. For example, component values of 0, 10, 150, or 255 are replaced with component values of 255, 245, 105, or 0, respectively. Implement the convertToNegative() method that performs this transformation.
